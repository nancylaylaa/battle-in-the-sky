# battle-in-the-sky
Deskripsi game
  Game Battle in the sky menggunakan pygame dari Python.
  Game ini dimainkan oleh 1 user/player yang menggunakan keyboard untuk control pesawat player.
  Game ini melatih kecepatan tangan dan konsentrasi user.
